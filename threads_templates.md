# Threads 글 유형별 작성 템플릿

> 알고리즘 신호 우선순위 기반: 댓글 > 골든타임 > 리포스트 > 저장 > 좋아요

---

## TYPE A — 의견형 (댓글·대댓글 유도)

**목적:** 대화를 길게 이어가기 → 알고리즘 1순위 신호 획득

**구조:**
```
[강한 주장 or 반직관적 사실 1–2줄]

[근거 or 경험 2–3줄]

[열린 질문으로 마무리]
```

**예시 템플릿:**
```
대부분의 유튜버가 썸네일에서 실패하는 이유는 클릭을 유도하려 해서다.

썸네일의 진짜 역할은 클릭이 아니라 "내 영상이다"를 각인시키는 것.
클릭률보다 재방문률을 높이는 썸네일이 채널을 키운다.

당신은 썸네일을 만들 때 뭘 가장 먼저 생각하나요?
```

**작성 체크:**
- [ ] 공감 or 반박을 유발하는 주장이 있는가?
- [ ] 질문이 "예/아니오"가 아닌 열린 질문인가?
- [ ] 200자 내외로 짧고 강한가?
- [ ] 게시 직후 내가 먼저 댓글 달 내용 준비했는가?

---

## TYPE B — 정리형 (저장 유도)

**목적:** 나중에 꺼내 보고 싶은 콘텐츠 → 알고리즘 3순위 신호 획득

**구조:**
```
[제목형 훅 — "~하는 N가지" or "~할 때 꼭 알아야 할 것"]

[리스트 or 단계별 정보]

["저장해두세요" CTA]
```

**예시 템플릿:**
```
유튜브 인트로에서 구독자를 잃는 패턴 5가지

1. 자기소개부터 시작 (시청자는 관심 없다)
2. "오늘은 ~에 대해 알아볼게요" 예고 반복
3. 배경음악이 목소리보다 크거나 비슷함
4. 첫 15초 안에 핵심 약속을 안 함
5. 인트로가 30초를 넘김

이 중 지금 당장 고칠 수 있는 건 4번이다.
나머지도 고치고 싶다면 저장해두세요.
```

**작성 체크:**
- [ ] 리스트·수치·단계 중 하나가 포함됐는가?
- [ ] 맥락 없이 봐도 이해되는 독립형 콘텐츠인가?
- [ ] "저장해두세요" or "북마크 추천" CTA가 있는가?
- [ ] 300자 내외로 읽기 부담 없는가?

## TYPE C — 리포스트형 (인용·리포스트 유도)

**목적:** 팔로워 외부로 확산 → 알고리즘 2순위 신호 획득

**구조:**
```
[한 줄 요약 — 리포스트됐을 때 단독으로도 강한 문장]

[3–4줄 부연 설명]

["리포스트" or "인용" 직접 요청 CTA]
```

**예시 템플릿:**
```
조회수는 알고리즘이 주는 것, 구독자는 내가 만드는 것.

유튜브에서 조회수가 갑자기 터져도 구독자가 안 늘면
채널이 아니라 영상 하나가 뜬 것이다.
알고리즘 의존도를 줄이려면 콘텐츠보다 캐릭터를 팔아야 한다.

공감되면 리포스트해주세요.
```

**작성 체크:**
- [ ] 첫 줄만 봐도 "이게 뭔 말이지?" 궁금증이 생기는가?
- [ ] 한 줄 요약 문장이 단독으로 의미를 갖는가?
- [ ] "리포스트해주세요" 직접 CTA가 있는가?
- [ ] 150자 내외로 더 짧고 강한가?

---

## 유형 선택 기준

| 상황 | 추천 유형 |
|------|----------|
| 영상 공개 당일 | C (리포스트형) — 확산 극대화 |
| 팔로워가 적을 때 | A (의견형) — 댓글로 관계 구축 |
| 정보성 콘텐츠 정리 | B (정리형) — 저장으로 신뢰 축적 |
| 논쟁적 주제 있을 때 | A (의견형) |
| 인플루언서와 교류할 때 | C (인용형) |

## 게시 직후 첫 댓글 템플릿

의견형:
```
저는 [내 경험/사례]에서 이걸 실감했어요.
여러분은 어떤 상황에서 느끼셨나요?
```

정리형:
```
이 중에 지금 당장 적용하기 가장 쉬운 건 [번호]번이에요.
어떤 게 제일 와닿으셨나요?
```

리포스트형:
```
특히 [핵심 포인트]를 경험하고 나서 생각이 바뀌었어요.
비슷한 경험 있으신 분 있나요?
```
